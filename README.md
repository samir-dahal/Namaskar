Every time you open a tab, you're greeted with a beautiful landscape photo from Nepal, mountains, valleys, temples, alongside the Bikram Sambat date, lunar tithi, public holidays and festivals, and current weather for your city.

> Whether you're Nepali living abroad, learning about Nepal, or just love the Himalayas, Namaskar keeps you connected to Nepal's culture and calendar every single day

### Features

- **Daily Nepal photo** — changes every day
- **Bikram Sambat date** with full Today's Nepali calendar info
- **Today's festivals, public holidays & tithi**
- **Live weather** for any city
- **English or Nepali (Devanagari)** display
- **Works offline** using the last cached data
- **No account, no tracking, no ads**

### Available on Chrome Web Store
Simply download from extension store and you're all done!
